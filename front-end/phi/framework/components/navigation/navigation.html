Inline Navigation w/Dropdowns

The default navigation includes the option for drop-down sub-menus. The first thing you need to do is declare inlineLinkList class on the ul for horizontal nav or linkList (see below) for a vertical one. Then, each main li gets a class navItem.

If you want sub-menus, then just add the class dropDownMenu on the top level ul, and any nested ul will automatically have the necessary functionality generated from the navigation.js plugin. Additionally, you can add hasHover to the top level ul to have a fallback CSS-only sub-menu functionality, if JS fails to load or execute.

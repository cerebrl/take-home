Typography

General Typography

Phi uses a typographic scale of 1.25 of the base font-size. Based on 1em or 16 pixels, that creates a 20px vertical unit. 20 pixels is easy to reason about and divide into smaller subunits. It may be a bit tight for a reading centric site or application, but for a interactive heavy one, this works great.

Header Styles

Optional Subheader Styles. If you have more than one header tag, make sure to an hgroup element. hgroup is being depracated, but I won't tell if you won't tell :)

To leverage a stacked head/subhead, you need to add a class of subheader to the lower header. This will remove the top padding to create a tighter relationship between the two headers.

If you want to inline a subhead, just use the small HTML element.

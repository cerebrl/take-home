Buttons

Basic Buttons

Many UI frameworks contain a plethora of button styles. We don't. We just start you off with the basic button. Just add the class button to anything clickable. By default, the button class does nothing more than add the necessary margin and padding (as you can see in the "No Style" example. If you want the default style, add the default class. It's up to you to take it from there.

Buttons come in different colors. Each color usually associates with an action. Use classes like primary, alert and the like to style them up.

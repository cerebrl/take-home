Tabs

Regular Tabs

Tabs are made of two objects: a ul list containing the tabs themselves, and a ul list containing the tab content. To use them, declare a class of tabs on the tab list ul and a class of tab on all contained li elements.

Then, you need to add a class of tabsContent to the content list ul. Finally, you need to make sure the href in the tab link matches the id of the tab content li.

Vertical Tabs

You can also use tabs in a vertical configuration by adding a class of tabsVertical to the ul.tabs element. These are great for handling an unknown or large number of nav elements.

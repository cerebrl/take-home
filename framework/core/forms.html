Need decorative/communicative elements tied to inputs? All you need to do is wrap the input element with the prefix/postfix element in a div tag. Help and Error States

Phi includes error states for labels, inputs and messaging that you can have your app generate programatically. All you need to do is add the "message" in a small element, add an error class to the wrapping element, and if you really want it to stand out, add an alert class to the "message".
